# Spectrometer
🔬 線上光纖光譜儀：探究實作操作指南Web-Based Fiber-Optic Spectrometer歡迎來到數位光學實驗室！這套系統能將你的電腦或平板，瞬間變成一台專業的「可見光吸收光譜儀」。在這裡，機器不是黑盒子，你將親自參與從「幾何定位」、「波長校正」到「訊號降噪」的每一個物理過程，並量測出真實的吸光度數據。🛠️ 第一階段：光學幾何與波長校正 (Calibration)我們必須先告訴軟體，相機畫面中的哪一個區域是光譜，以及每個畫素對應的「真實波長」是多少。請準備一支標準三基色螢光燈管作為校正光源。步驟 1：鎖定取樣區域 (ROI)切換至 「1. 光纖幾何與波長校正」 分頁。點擊 [啟動攝影機]，將光纖對準螢光燈管。調整 寬度 (W)、厚度 (H) 與 旋轉角度 拉桿。💡 科學小提示：請讓綠色的取樣框「精準且水平地」包住光譜彩條最亮的中心區域，並避開上下邊緣的暗區，這樣能獲得最佳的訊號品質。步驟 2：進階濾波處理 (DSP 降噪) - 非必填，但強烈建議相機感光元件會產生熱雜訊，為了追求極致精準，我們可以先「扣除背景」。完全遮蔽光纖頭（確保沒有任何光線進入）。點擊 [🌑 擷取暗場]，系統會連拍 30 幀全黑畫面取平均，計算出儀器的絕對底噪。點擊 [扣除背景: 開] 與 [平滑濾波: 開]，你會發現曲線瞬間變得平順且乾淨。步驟 3：特徵波長絕對對齊利用螢光燈管中已知且恆定的原子發射線，將影像畫素映射為物理波長。點擊 [📍 開始標定特徵波長]。標定汞藍線 (435.8 nm)：在圖表上尋找藍色區塊中最尖銳的波峰，點擊它。（可開啟 [🧲 磁吸對位] 讓系統自動抓取最高點）。依序標定 汞綠線 (546.1 nm) 與 銪紅線 (611.6 nm)。完成後，系統會利用二次多項式擬合，自動換算出全波段的正確座標！🧪 第二階段：吸收光譜對照掃描 (Absorbance Scan)校正完成後，我們就可以開始量測未知溶液（例如硫酸銅、色素水）的光學特性了！步驟 4：擷取基準對照光源 ($I_0$)切換至 「2. 吸收光譜對照掃描」 分頁。將光纖對準你實驗用的主光源（例如白光 LED、鹵素燈），中間放上裝有純水（空白液）的試管。觀察上半部的圖表，確認光譜強度穩定後，點擊 [📸 拍照定格]，接著點擊 [✅ 儲存為基準光源]。💡 科學小提示：這個步驟非常重要！它記錄了光源原本的強度分佈 ($I_0$)，我們才能知道後續被樣本「吸收」了多少光。步驟 5：待測樣本掃描 ($I$) 與吸光度轉換 ($A$)將純水試管替換為裝有待測溶液的試管。點擊下半部區塊的 [啟動攝影機]，你會看到穿透樣本後剩餘的光譜強度 ($I$)。點擊 [切換：顯示吸光度]！💡 物理心法：系統此時正利用比爾-朗伯定律公式 $A = -\log(I/I_0)$ 進行即時運算。圖表曲線越高，代表該波長的光被吸收得越徹底！步驟 6：設定有效光源閾值 (SNR Mask)如果你發現吸光度曲線在某些波段（例如深紅光區）出現瘋狂跳動的雜訊，請不要慌張！這代表你的基準光源在該波段太暗了，微弱的光線被相機底噪給淹沒。請微調 [有效光源閾值] 拉桿（例如設為 10 Counts）。系統會非常聰明地過濾掉這些雜訊無效區，強制歸零，還給你一條完美平滑的真實吸收曲線。📊 第三階段：資料紀錄與輸出 (Export)科學實驗講求證據與重現性，當你捕捉到最完美的曲線時：🖼️ 儲存圖表：點擊各區塊右上角的按鈕，系統會自動將「光譜彩條」與「數據圖表」完美合成一張帶有時間戳記的高解析度 PNG 圖片，這是科展海報上最棒的視覺呈現。📥 匯出 CSV：點擊下半部的匯出按鈕，系統會打包所有波長對應的「校正強度、基準光 $I_0$、樣本光 $I$、吸光度 $A$」原始數據。你可以將它丟進 Excel 或 Python 進行更深入的濃度定量分析！
